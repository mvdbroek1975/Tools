   In order to use this app, create a free account on https://pushover.net/login. Then register this app and set the userkey and apptoken 
   in the config file. Download the pushover app on you phone and login. Now your ready to receieve push messages.
   The defaults are:
     - interval is set to 5 minute checking
     - currency threashold is 200 euro's. 
     - currency is set to eth (Eutherium) but you can also use another currency like btc (Bitcoins)
